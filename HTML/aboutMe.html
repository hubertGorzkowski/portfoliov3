My road

It all started with me studying for my first school test for web development lessons. Those were the basics of the basics.

At the end of the school year, I had to create my own website. I liked it so I started doing courses on the udemy platform in the basics of HTML and CSS. The entire course has been divided into several smaller segments. After doing the first two, it's time for JavaScript.

After learning the basics of the language and making some of my own, more difficult projects for me, the time has come to choose a library.

I chose React. Currently i did some projects in it like form validation and ToDoApp.

Currently, I watch a few YouTube channels that show me cool ideas to stylize my website. Recently I am also doing an advanced course in CSS and JavaScript projects.

The course covers technologies such as Ajax, npm, bootstrap, canvas and basics of Node.

After finishing this course, I would like to learn more technologies like more advanced things in Node, Express framework and mongoDB.

Up to this point, I've learned HTML, CSS, BEM methodologies, SASS preprocessors, JavaScript, AJAX, github and the basics of React. Due to the fact that I went to the IT technician profile many things did not cause me problems, such as using the console and commands.

I am currently working as a fiber optic installer. Due to work, I'm only home on weekends, so I don't have much time to study. Despite everything, I'm still learning because it gives me a lot of joy and I would like to work as a front-end developer in the future.
